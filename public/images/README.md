## 图片格式

- 支持: JPG, PNG, WebP
- 推荐: JPG（文件小，加载快）
- 文件大小: 建议每张 < 500KB

## 图片优化

建议使用在线工具压缩图片：
- https://tinypng.com
- https://squoosh.app

## 注意事项

- 如果图片不存在，系统会自动使用占位图
- 图片路径是相对于 `public` 目录的
- 修改图片后需要重新构建项目
